# KOMERZ - Marketplace Multilingua 🚀

**KOMERZ** è una piattaforma di compravendita di oggetti usati moderna, veloce e intuitiva. Il progetto è focalizzato sulla facilità d'uso per i venditori e su un sistema di revisione sicuro per garantire la qualità degli annunci.

---

## 🎨 Palette & Design
Il progetto segue un'identità visiva decisa:
* **Primary:** `Indigo` (Profondità e stabilità)
* **Accent:** `Amber` (Energia e richiamo all'azione)

---

## 🛠️ Tech Stack
* **Framework:** [Laravel 13](https://laravel.com)
* **Frontend Dinamico:** [Livewire 3](https://livewire.laravel.com)
* **Database:** MySQL / PostgreSQL
* **Styling:** Bootstrap 5 / Sass
* **Autenticazione:** Laravel Fortify
